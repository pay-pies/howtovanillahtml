What is HTML & CSS?

HTML stand for Hypertext Markup Language and CSS stands for Cascading Style Sheets. HTML is used to structure a web page and its contents. CSS is used to add style and describe the presentation of a markup language like HTML. The HTML will go in the body tag and CSS will go in the style in the head.

Vanilla HTML/CSS is the plain language without any extra plugins or fancy frameworks.

Text tags.

There are 6 'h' sizes: h1, h2, h3 ... h6. h1 being the largest and h6 being the smallest. h1 is typically used for titles, h2 for headings, h3 for sub-heading etc. There is also 'p', which is your normal paragraph text.

There are 4 text tags you need to know: br, em, u and strong. The br tag is the line break tag. You will only need one of these unlike most tags. 'em' is the italic tag, which makes your text italic (wow! i know right). 'u' is the underline tag and 'strong' is the bold text tag.

Lists

There are two types of list: ordered and unordered. Ordered lists can be ordered like a numbered list, a list that has roman numerals or the alphabet etc. An unordered list is a list that is just a list with bullet points or something. For ordered you use the 'ol' tag and for each item in the list you use the 'li' tag.
